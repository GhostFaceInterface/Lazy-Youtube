her şeyden önce chromedriver'ın yüklü olduğuna emin olun.
repoyu klonladıktan sonra main.py dosyasının bulunduğu dizine gelin.
şu komutları yazın : 
1- pip install virtualenv
2- python -m venv env
bu komutlar sanal ortamı oluşturacak. 
ardından sanal ortamı çalıştırmak için 
env\Scripts\activate komutunu çalıştırın.
üstteki aşamada önceliklidir. Bu işlemler yapılmadan alttaki işlemlere geçilmez

sanal ortam oluşturulduktan sonra bağımlılıkları indirmek için :
pip install -r requirements.txt

bağımlılıklar indirildikten sonra main.py değil watchdog_runner.py scripti çalıştırılmalı.
yani python wachdog_runner.py komutunu çalıştırın.

çalıştırdıktan sonra mutlaka hata alacaksınız. Olası hatalar : 
1- chromedriver konumu : mac bilgisayarda chromedriver çağırmak için bir konum belirtmeme gerek yok. chromedrivernerede yüklü ise onu bulur ve getirir. 
windows için bu süreç nasıl işler bilmiyorum ona bakmak gerekiyor. chromedriver konumu bulunamadı vs gibi hatalar alıyorsanız
